Work

data and activism

Playful learning resource about digital security for activists

Team: Amber Macintyre, Leil Zahra

As part of my work at tactical tech, I worked on conceptualizing and creating the "data and activism activity book". The project uses playful exercises to make people reflect on their data practices and learn about security aspects of the same. I helped create the concept and content for a first prototype, and designed teaser cards as well as the final publication. The visual aesthetic for the project borrows from scrapbooking and activity book elements as well as the "data and activism" identity colors and existing collage-style artwork.
